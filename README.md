# DecodeLabs Task 3 - Banking Management System

## Description

A Java Swing based Banking Management System developed with a modern Graphical User Interface (GUI). The application allows users to register, log in securely, perform banking operations such as deposits and withdrawals, view account balances, and maintain transaction history.

This project was developed as part of the **DecodeLabs Internship Program** to demonstrate Java programming, Java Swing GUI development, Object Oriented Programming (OOP), and event driven application design.

## Features

- Secure User Login
- Account Registration
- Deposit Money
- Withdraw Money
- Balance Visibility Toggle
- Transaction History
- Modern and User Friendly GUI
- Object Oriented Programming (OOP) Design
- Easy Navigation and Interactive Interface

## Technologies Used

- Java
- Java Swing
- NetBeans IDE
- Object Oriented Programming (OOP)

## How to Run

1. Open the project in NetBeans IDE.
2. Build the project successfully.
3. Run the application.
4. Register a new account or log in.
5. Perform banking operations such as deposits, withdrawals, and viewing transaction history.

## Learning Outcomes

This project helped in understanding:

- Java Swing GUI Development
- Event Handling
- Object-Oriented Programming (OOP)
- User Authentication
- Banking Transaction Logic
- Java Application Development

## Project Purpose

The purpose of this project is to simulate a basic banking system through a desktop GUI application while practicing Java programming concepts, GUI design, and OOP principles.
